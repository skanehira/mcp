# mcp
Copy multiple files using your `$EDITOR`.
This is inspired by [mmv](http://github.com/itchyny/mmv)

![](https://i.imgur.com/2D9S6WW.gif)

## Usage
```sh
# copy all of current directory's files
$ mcp *

# copy file
$ mcp file
```

## Installtion
```sh
$ go get github.com/skanehira/mcp/cmd/mcp
```
